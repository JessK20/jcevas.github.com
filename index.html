Proyectos Realizados

Project - Publicorp

Es una empresa donde brinda servicios de impresiones y paneles, entre otras anuncios publicitarios. En este proyecto se uso AJAX, para el manejo asíncrono de las imágenes y que vean una galeria amigable en la parte de servicios (:

Project - Sile

Sile es una empresa que se dedica al transporte que contribuye al comercio tanto externo como inteno. En este proyecto se uso UX para la version mobile ya que por el requerimiento del cliente, se enfocara el usuario en llamar a esta empresa, logrando asi que sus servicios sean mas requeridos.

Project - Dmotiff

Dmotiff es una tienda online de personal shopping, donde se enfoca en que cada usuario o cliente se sienta comodo en escoger el producto y obtenga una compra segura e satisfactoria.

Project - Británico

Britanico una institucion de enseñanza en idiomas, hemos rediseñado la web para que tenga sea mas vistosa y agradable, logrando alzar que hayas mas estudiantes matriculandose online.

Project - Inkafarma Selección

Inkafarma, aqui se centro más en que se publicaran oferta de trabajo de cualquier rubro, similar como las plataformas virtuales como computrabajo, entre otros. En este proyecto se plasmo que al momento que el usuario digitara sus datos fuera asincronamente y que se guarden automaticamente, es decir, siendo mas usable.

Project - Llantilandia

Es una empresa de respuestos y venta de llantas, para cualquier tipo de medio de transporte.
